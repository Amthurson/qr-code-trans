## 📋 项目概述

一个完全离线的问卷数据传输系统，通过二维码编码实现患者问卷数据的安全传输，无需网络连接。

### 核心功能

- ✅ **患者端**：问卷填写、实时体积计算、编码压缩、二维码生成
- ✅ **医院端**：手动输入/扫码、解码还原、数据展示
- ✅ **核心算法**：多选题三种编码策略自动选择、LZ 压缩、体积分析
- ✅ **容量控制**：实时反馈、字数限制、超限预警

## 🔧 核心算法

### 多选题编码策略

```typescript
// 自动选择最佳策略
策略选择逻辑：
1. 计算组合总数 = C(n,1) + C(n,2) + ... + C(n,maxSelect)
2. 列表编码长度 = 已选数量 k
3. 位图编码长度 ≈ n * log₂ / log₃₆
4. 组合映射：
   - 组合数 <= 36 → 1 字符
   - 组合数 <= 1296 → 2 字符
   - 组合数 > 1296 → 禁用
5. 选择最短的策略
```

### 编码格式

```
V2|P:患者 ID|A:年龄|S:性别|1:SA|2:MB1F|3:Txxx|4:MLAB|...

标记说明：
- V = 版本
- P = 患者 ID
- A = 年龄
- S = 性别
- S = 单选答案
- N = 数值答案
- T = 文本回答
- ML = 多选 - 列表
- MB = 多选 - 位图
- MC = 多选 - 组合
```

### 压缩流程

```
原始 JSON → 结构化编码 → LZ 压缩 → URI Encode → 二维码
```

典型压缩率：**70-80%**

## 📝 示例问卷

当前 Demo 使用**脊柱侧弯预问诊问卷**：

1. 性别（单选）
2. 年龄（数值）
3. 身高（数值）
4. 体重（数值）
5. 家族史（单选）
6. 症状（多选，最多 5 项）
7. 确诊情况（单选）
8. Cobb 角（数值）
9. 治疗方式（填空，40 字）
10. 主要不适（问答，200 字）

**典型数据体积**：
- 原始 JSON: ~800 字符
- 编码后：~400 字符
- 压缩后：~200 字符
- 压缩率：~75%
